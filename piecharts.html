Pie Charts

Version 1

The first version is a very basic pie chart with data values.

Version 2

Version 2 features a labels, display text, and colors that correlate to the type of pie

Version 3

Version 3 includes added interactivity, using the hoverBorderColor, hoverBorderWidth, and hoverBackgroundColor tooltips. When a user hovers over a specific segment of pie, the segment will slightly change color and change the border width to give an interactive feel. The title text was also enlarged.
