Using this cell, please write a short, clear paragraph explaining how you selected and tuned your model for the project. I chose knn as the classifier for this dataset for a few reasons. I suspected that wines with similar characteristics would be rated as having a similar quality which is a 1-10 rating. If this was the case, knn would be an effective classifier due to clustering of wines with similar characteristics. I also liked that knn only has 1 hyperparameter, which is the value for k, or the number of neighbors. A reason not to chose knn for this dataset would be that there are 11 features, and knn can struggle with high dimensionality. I chose to go full steam ahead with knn anyway.

Initially, I am going to include all features without engineering any of them to see how knn performs on all features, just for fun.

#Instead of 1-10 quality as labels, I am consolidating into 4 labels: poor, fair, good, excellent
rating = []

    if quality<4:
        rating.append("Poor")
    elif quality>7:
        rating.append("Excellent")
    elif quality in range(4,6):
        rating.append("Fair")
    else:
        rating.append("Good")
